Motor Speed Control

Design and implementation of a motor speed control system for transporting a dynamically unstable load.

Introduction

The goal of this project was to design a system that transports a vertically standing 1-foot long aluminum bar across a variable distance (5â€“10 ft) without toppling. The challenge was to minimize transport time while ensuring the bar remained stable at the end of motion.

Theory

The system's stability is governed by the moment equation:

a = (g * w) / h

Where:

  • a = Maximum allowed acceleration
  • g = Gravitational acceleration
  • w = Width of the base
  • h = Height of the bar

Simulation

A dynamic simulation was performed using SolidWorks to validate the calculated maximum acceleration and to determine the optimal motion profile for stable transport. Simulation results confirmed that the derived stability condition matched the system's physical behavior.

Prototype Construction

The physical prototype was built using the following components:

  • 12V gear motor
  • Arduino Uno board with power amplifier
  • Aluminum extrusion bar
  • Foam board base
  • Custom 3D printed wheels and mounts

Code Development

The control code was developed using Arduino IDE. The system is programmed to adapt to different travel distances between 5 and 10 feet. The control loop adjusts motor speed based on position feedback to ensure stable motion and minimize transport time.

Testing

The prototype was tested over various travel distances. Adjustments were made to motor speed and control loop parameters based on real-time performance. The final system demonstrated stable transport of the bar without toppling and consistent stopping behavior.
