Automated Trash Container Design

Design and development of an automated trash container with a linkage-driven tray system.

Project Overview

This project involved designing a motorized trash container with a tray that automatically lifts and deposits trash into a container opening at the back. The system was modeled using **SolidWorks** and motion-tested using **Math Illustrations** software.

Design Process

The design focused on creating a smooth tray motion using a four-bar linkage system. The pivot points were carefully positioned on the sides of the container to allow for consistent lifting and dumping without spilling the contents. The system was simulated to verify motion accuracy and tray positioning.

Challenges and Solutions

A key challenge was ensuring that the tray followed a consistent lifting path without hitting the container edges. This was resolved by adjusting the linkage lengths and pivot positions in Math Illustrations until smooth motion was achieved.
